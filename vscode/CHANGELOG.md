# Changelog

## 1.1.0

- Quick Paste command (no prompts, uses all defaults)
- Upload file from explorer sidebar (right-click)
- List My Pastes — browse and open recent pastes
- Delete Paste by slug
- Remove API Key command
- Default expiration setting (never, 10m, 1h, 1d, 7d, 30d)
- Clipboard copy toggle
- Status bar visibility toggle
- Skip title/visibility prompts option
- Line numbers and file info header options
- Max content length with truncation warning
- Notification style: full, minimal, or silent
- Custom domain support via instance URL

## 1.0.0

- Initial release
- Create paste from selection or file
- Open paste by slug/URL
- Visibility picker (Public, Unlisted, Private)
- Secure API key storage
- Status bar button
- Right-click context menu
